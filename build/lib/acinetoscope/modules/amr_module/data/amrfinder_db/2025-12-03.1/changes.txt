----------------------------------------------------------------------
Changes included in 2025-12-03.1
----------------------------------------------------------------------

This is a major release of both the AMRFinderPlus software (v.4.2.4) and database (2025-12-03.1).

The AMRFinderPlus 4.2.4 software release includes:
- New POINT_DISRUPT feature to identify lesions in genes that cause resistance when they are disrupted.
- Small in the point mutation nomenclature for nonsense mutations, frame shifts, and deletions.
- Refactored BLAST interpretation code and small changes that will slightly improve some alignments.
- StxTyper 1.0.45, See https://github.com/ncbi/stxtyper/releases/tag/v1.0.45 for the differences.

See https://github.com/ncbi/amr/wiki/New-in-AMRFinderPlus for more information.


The AMRFinderPlus 2025-12-03.1 database release updates include:

- The new "POINT_DISRUPT" functionality has been added for Klebsiella_pneumoniae ompK35, ompK36, and cirA genes. Putatively function disrupting mutations not in the database will be returned in addition to the published mutations in these genes already included in the AMRFinderPlus database. We will be expanding the number of organisms that use this functionality in future database releases.
- Point mutation nomenclature for deletions and stop codons has changed to more closely mimic HGVS standards. Please see https://github.com/ncbi/amr/wiki/Point-mutation-nomenclature for details.
- Several records had their reference sequences withdrawn from GenBank. We replaced them with other sequences when available. We also replaced sequences based on a review where we identified incorrect start sites for a few proteins. You will see them in the detailed list below as proteins removed and added.
- Changed reference for Staphylococcus aureus mprF from WP_111724484.1 to WP_001071135.1 to improve detection of mprF_V351E daptomycin resistance point mutation. The previous reference had a rare SNP in the immediately adjacent position 352
- Investigated and removed tet(34) based on a rereading of the original paper, AST results available in Pathogen Detection showing no effect on phenotype, and a review of the comparative genomics.
- Fixed a typo to standardize the blaOXA-MSI protein name (was class D beta-lactamase MSI-OXA, now class D beta-lactamase OXA-MSI).
- Fixed a typo in Gene family symbol blaCIB was corrected to blaCIM. The only currently named allele, blaCIM-1 was correctly named.
- New Helicobacter pylori beta-lactam and quinolone resistance point mutations.
- Further collaboration with Gen-FS Salmonella Virulence Factor Working group members resulted in an additional 58 virulence genes for Salmonella building upon the 8 added earlier (Harhay et al., 2024 - PMID: 39845055).
